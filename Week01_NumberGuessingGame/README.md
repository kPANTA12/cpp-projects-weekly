# Number guessing game (c++)

This is console based game.
It generate the number between 1 and 100 , and the player guess it.
After guess program tells wheather the guessed number is too high or too low or correct.

When the player guess is correct it tell the number of tries, also it ask player if player wants to play or exit from game.

-----------------------------
## Features
-Generate a random number between 1 and 100
- Gives feedback after the guess (either it is high,low or correct)
- Gives number of attempts
- Option to play again

--------------
## How to run 
1. Clone the project
2. Complile the program in IDE
3. Play game in console and enjoy
